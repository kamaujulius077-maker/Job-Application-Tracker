# Job-Application-Tracker
This is a website that solves the problem of loosing track where you've applied, interview stages and follow-ups during a job seearch

## Description 
Track job applocation using JavaSript constructors, prototypes, and TDD

## Features
Add new job with company name, position title, date apllied and status
Filter by status: All, Aplied, Interviewed, Offered and Rejected.
Application Statistics: Total Applications, Applied, Interviewed, Offered, Rejected and Success Rate.
Data persists with localStorage
